# How to use

## Install docker

1. Installera docker desktop på datorn
2. Starta docker desktop
3. gå in till visual-map repository via kommandotolken.
4. kör `docker build -t nextjs-docker .`
5. kör `docker run -p 3000:3000 nextjs-docker`

## Kör den lokalt

1. Installera docker desktop på datorn
2. Starta docker desktop
3. gå in till visual-map repository via kommandotolken.
4. `npm run dev`
